    • What is Mathematics?

    Mathematics is in many ways the most rewarding A level to study; it opens a wide-range of career opportunities from investment banking to research in the sciences. However, many people find mathematics hard as it requires you to link many separate rules together and apply them to answer a question, as a result Mathematics requires practice and hard-work. The textbook starts with the assumption that you have passed a GCSE Maths course, although some principles will be briefly reviewed.

    Mathematics is simply a set of rules which make up a game. The most basic rules are taught from primary school and gradually built upon so that you can solve more problems and interact with mathematics in a more creative, abstract way. Like any game some are naturally better at certain aspects than others, regardless of this Mathematics is useful for any individual as once you have learnt enough rules you can apply them to the real world to solve real-life situations. However, another side of Mathematics lies in exploration of the game; in creative, some may even say beautiful, explorative ways to find and establish new rules which enrich the toolkit Mathematicians already have - Welcome to Pure Mathematics.

    How do Mathematicians find rules?

    The job of a pure mathematician lies in discovery; discovering new rules of patterns that lie in wait for the most intelligent of beings to seize them as a result of the basic assumptions, more formally known as axioms, that define the game of mathematics. Let our journey begin by defining some basic assumptions:

    • There exists a set (collection of individual objects) of positive integer numbers known as the Natural Numbers: \( \N = \{ 1,2,3,4,... \} \)
    • There exists a set of integer numbers including all natural numbers, \( \N \), 0 and negative integers: \( \Z = \{ ...,-1,0,1,... \} \)
    • Letters can be used as variables; to represent different types of numbers or a specific number (Basic algebra)
    • There is a set of numbers known as Rational numbers which can be expressed exactly as a ratio (fraction) of two integers: $$ \text{Q} = \{\frac{p}{q} \text{where p and q are integers and } q \neq 0 \} $$
    • There exist the following basic operations (and their inverse operations): Addition, Subtraction, Multiplication, Division
    • An even number is defined as a multiple of 2, \( 2n \), and an odd number is defined as not an even number: \( 2n - 1, 2n + 1\)

    Methods of Proof

    Now we can go about proving, or disproving, new rules given the above assumptions. The methods we will learn and need to apply to basic problems are firstly proof by deduction. This is when we manipulate pre-established rules to find new rules. As an example, we can prove that an even number added to a different even number is even. let \(2n\) represent an even number and \(2m \) represent another even number: $$ 2n + 2m = 2(n + m) $$ Since we can factor out 2 this shows that number \( n + m \) is even. Another method of proof, or more accurately disproof, is by counter-example: find one example that contradicts the new rule. For example we can disprove that for 2 integers, \( a,b \), if \(a^2 = b^2 \) then \(a = b\): If \(a = 3, b = 3 \) then the statement holds. However if \(a = 3, b = -3 \): $$ 3^2 = (-3)^2, 3 \neq -3 $$ Then the statement does not hold, therefore we have proved the statement to be false. QED. Another method of disproof is by assuming the statement to be true and then finding a contradiction in previously accepted rules of mathematics as a result of the assumption, which proves that the assumption made to be false. A famous proof by contradiction is the proof that \( \sqrt{2} \) is irrational:

    1. Assume that \( \sqrt{2} \) is a rational number and therefore can be expressed as \( \frac{a}{b}, b \neq 0 \) where \( a, b\) are integers in their simplest terms (they do not both have a common divisor).
    2. \((\sqrt{2})^2 =(\dfrac{a}{b})^2\ \\ 2 = \dfrac{a^2}{b^2} \\ 2a^2 = b^2\)
    3. We have found that \( b \) must be even as \( b^2 \) is even and the square of an even number is an even number.
    4. We can apply the same argument to \( a \) which must also be even as a result. We can rewrite b and a as \(2m, 2n \) (multiples of 2).
    5. Therefore \( \sqrt{2} = \dfrac{2m}{2n}\) which contradicts the fact that \(a, b\) are in their simplest terms. Therefore by contradiction \( \sqrt{2} \) must be irrational. QED.

    The final method of proof we must know is proof by exhaustion. This is when we work out every possible option to be true so therefore the statement must be true. For example we could prove by exhaustion all of the prime numbers (only exactly divisble by 1 and itself) between 1 and 100. Proof by exhaustion is also helpful in proving more complex problems by splitting them up into smaller proofs, such as proving the statement true for all even numbers then for all odd numbers and therefore all numbers. Finally, whilst all the proofs above have been with numbers, we can also apply the same methods of proof to geometry (shapes and graphs) yet through working with a different set of established rules.

    Simple Questions

    Here are some simple proof problems with solutions. Whilst reading through this textbook, each rule taught has a proof, and we recommend you to try and prove each rule to help practice this skill. The questions presented are remedial since assuming you have just started the A level course and have not been taught any of the mathematical techniques to prove harder problems.

    Warm-up
    1. Prove that an even multiplied by any integer will be even.
    2. Prove that an odd multiplied by an odd number is odd.
    3. Prove that \( n^2 - n \) is always an even number.
    4. Prove the sum of 2 consecutive prime numbers is always even.
    5. Given that a is a positive real number, prove that: \( a + \dfrac{1}{a} \geq 2\)
    Warm-up question solutions
    1. An even integer n is a multiple of 2 so can be written as \(2k\). Let m represent any integer.
      \(nm = 2km = 2(km)\).

    2. Let \( 2j + 1 \) represent an odd number, n, and let \(2k + 1\) represent another distinct odd number, m.
      \(nm = (2j + 1)(2k + 1) = 4jk + 2j + 2k + 1 = 2(2jk + j + k) + 1\). By substitution of \( 2jk + j + k = z\)
      \(nm = 2z + 1\) which proves the statement.

    3. \( n^2 - n = n \times n - n = n(n - 1) \). Firstly let us prove the statement for odd numbers then even numbers. If n is odd then let \( n = 2k + 1\):
      \(2k+1(2k + 1 - 1) = 2k+1(2k)\). Since anything multiplied by an even number is an even number we can stop here. If n is even then let \(n = 2m \):
      \(2m(2m - 1).\) The same logic applied above can be used here. QED.

    4. We can disprove this given that 2 and 3 are both consecutive prime numbers and sum to an odd number; 5.

    5. Again, we can also disprove this statement given that \( a > 0\) and if we let a be 1 it is less than 2.
