    • Power laws

    The purpose of the Algebra module is to teach you new methods to manipulate and solve problems involving letters. Later, we use these methods to explore the relationship between an input and output variable on a graph, as a function.

    Let us first consider what a "power", exponent or index (whatever term you may prefer) actually means: $$ a^n = a_1 \times a_2 ... \times a_n $$ As we see above "a to the power n" means "a times a, n times." There are various rules that allow us to simplify these terms if the base, a, is the same:

    1. \(a^n \times a^m = a^{n + m}\)
    2. \(a^n \div a^m = a^{n - m}\)
    3. \( (a^n)^m = a^{n \times m}\)

    Given we have defined the power rules for positive integer powers, how may we extend these to other number sets? What would \( a^0 \) be? Well intuitively you may think that it must be 0 as it would mean "a times a, 0 times". However, in mathematics, it is very important for rules to be consistent so we must define \( a^0 \) as 1 so that: $$ \dfrac{a^1}{a^1} = a^{1 - 1} = a^0 = 1 $$ Now, how would we extend our understanding of power rules to negative numbers? Let us examine an example; if \( a \neq 0 \) and \( m < n\): $$ \dfrac{a^{n-m}}{a^n} = a^{(n-m)-n} = \dfrac{a_1 \times a_2 \times ... a_{n-m} }{a_1 \times a_2 \times ... a_m } = \dfrac{1}{a^m} $$ We can cancel each term on the top and bottom by a the maximum number of times possible and since there are fewer a terms on the top we would be left with 1 and the remaining \( a^m \) on the bottom. Therefore for negative powers we have the rule: $$ a^{-m} = \dfrac{1}{a^m} $$ Now we are ready to extend these rules to include all rational numbers (ie fractions). But first we must remind ourselves of what roots mean. A square root of a number, a, is the number which when multiplied by itself once gives the number a exactly; the nth root of a number, a, would be a number which when multiplied by itself n times gives the number a exactly. Some square roots are integers but a lot are irrational and are left in their root or surd form \( \sqrt{a} \). Using power rule 1. :

    $$ a^n \times a^{\tfrac{1}{n}} = a^{\tfrac{1}{n}} \times a^{\tfrac{1}{n}} ... \times a^{\tfrac{1}{n}} = a^{\tfrac{n}{n}} = a $$ If we substitute \(a^{\tfrac{1}{n}} = b\) we arrive at \( b^n = a \) so then we must define \(a^{\tfrac{1}{n}} = \sqrt[n]{a} \). Finally, we end this section by generalising this for cases where the numerator is not 1 but instead \(m\): $$ a^{\tfrac{m}{n}} = (a^{\tfrac{1}{n}})^m = \sqrt[n]{a^m} $$

    Surd rules and rationalising the denominator

    With the power laws we introduced roots, Surds are simply irrational roots. Here are some rules for simplifying surds (and roots):

    1. \((\sqrt[n]{a})^n = a\)
    2. \(\sqrt{ab} = \sqrt{a} \sqrt{b}\)
    3. \( \sqrt{\dfrac{a}{b}} = \dfrac{\sqrt{a}}{\sqrt{b}} \)
    4. \(c \sqrt{a} + d \sqrt{a} = (c + d)\sqrt{a} \)

    Sometimes when dealing with a problem, we want to frame it in a certain way or rewrite it into a certain form; this is not necessarily simplifying the problem; as is the case for rationalising the denominator. This means making the denominator contain only rational numbers and moving irrational numbers (surds) to the numerator. We can easily do this for a simple fraction such as \( \dfrac{1}{\sqrt{a}} \) by multiply top and bottom by \(\sqrt{a} \). However what about rationalizing \(\dfrac{1}{\sqrt{a} - b}\)? In order to do this recall algebraic identities involved in expanding double brackets:

    • \( (A+B)^2 = A^2 + 2AB + B^2 \)
    • \( (A-B)^2 = A^2 - 2AB + B^2 \)
    • \( (A+B)(A-B) = A^2 - B^2 \)

    The last identity is the difference of two squares and allows us to rationalize the denominator for more problems such as with the above example: $$ \dfrac{1}{\sqrt{a} - b} = \dfrac{1(\sqrt{a} + b)}{(\sqrt{a} - b)(\sqrt{a} + b)} = \dfrac{\sqrt{a} + b}{a - b^2} $$

    Quadratic functions

    From GCSE we should know that a quadratic is a polynomial with the general form: $$ ax^2 + bx + c = 0 $$ We should also be aware of functions which take a set of inputs (domain) and map them to a set of corresponding outputs (range). \( f(x) = x^2 + 2\) is an example of a quadratic function where \( f(x) \) means 'f of x' and represents the outputted value. Another way of thinking of this is \(f(x)\) is the dependent variable which depends on the value of \(x\). Often, \( f(x) = y\). As a rule of thumb, the dependent variable is plotted on the y-axis (vertical line), and the x-axis is the independent variable (horizontal).

    What do quadratic graphs look like? Well they all have a general curved shape known as a parabola. The best way to learn about graphs, is to play around with them so we recommend you visit desmos and investigate them yourself before continuing. Here is the graph of \( y = 2x^2 + 4x + 1 \):

    From careful observation we see that the parabola crosses the x-axis twice and the y-axis once. Also the curve seems to be symmetric about its' minimum point. If we were able to find these points then we could sketch any quadratic graph:

    1. points on the x-axis (the horizontal line with equation \(y = 0\)) can be found by solving \( f(x) = 0 \) for values of \(x\). These values are often known as the solutions to the equation or the roots of the function.
    2. The y-intercept (y-axis has equation \( x = 0\)) can be solved by finding the value of \(f(x)\) from the quadratic equation when \(x = 0\).
    3. The minimum point is a stationary point and has a gradient of 0. Stationary points for parabolas have the greatest or least y value so we could find it this way alternatively. A final method would be finding the x-coordinate half-way between the 2 roots of the function and then plugging this value into the equation to find the corresponding y value.

    If the above picture of a parabola has a minimum stationary point, what would the shape of a parabola with a maximum point look like? Why would it have a maximum point? Well the shape of the parabola is determined by the whether the leading coefficient (the number term, a in front of \(x^2\)) is positive or negative; positive is a happy-faced parabola, negative a sad one. From the general form of a quadratic we can derive a formula to find us the coordinates for the stationary point of a parabola: (Try and have a go deriving it!) $$ ax^2 + bx + c = a(x + \dfrac{b}{2a})^2 + (c - \dfrac{b^2}{4a}) $$
    It is important here to consider that the coordinates of the turning point for equation \( a(x + p)^2 + q\) (substituting p and q in place of the whole completing the square formula) are \( (-p, q) \). Why this is the case will soon be evident once we consider graph transformations of functions.

    Now we have found a way to find the stationary point of a parabola, and finding the y-intercept is straight-forward, how do we go about finding the roots of a quadratic equation or more precisely, solving \( ax^2 + bx + c = 0\) for x. Well, we can factorise (using methods assumed to be taught to you at GCSE) and solve. But sometimes quadratics don't factorise nicely. We can derive a formula to solve quadratics (incidently known as the quadratic formula) by rearranging the general quadratic equation in terms of x to arrive at: $$ x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a} $$

    How many solutions does a quadratic equation have? Well from the quadratic formula if we treat \( \sqrt{b^2 - 4ac} \) as "something" or z we end up with either 2 none or 1 depending on the value of \( b^2 - 4ac \); this is known as the discriminant and the following rules tell us how many real solutions a quadratic equation has (or how many times the curve passes the x-axis):

    • When \( b^2 - 4ac > 0\) it has 2 real solutions
    • When \( b^2 - 4ac = 0\) it has 1 repeated real solution
    • When \( b^2 - 4ac < 0\) it has no real solutions

    Simple Questions

    Here are some simple proof problems with solutions. Whilst reading through this textbook, each rule taught has a proof, and we recommend you to try and prove each rule to help practice this skill. The questions presented are remedial since assuming you have just started the A level course and have not been taught any of the mathematical techniques to prove harder problems.

    Warm-up
    1. Prove that an even multiplied by any integer will be even.
    2. Prove that an odd multiplied by an odd number is odd.
    3. Prove that \( n^2 - n \) is always an even number.
    4. Prove the sum of 2 consecutive prime numbers is always even.
    5. Given that a is a positive real number, prove that: \( a + \dfrac{1}{a} \geq 2\)
    Derivation of Completing the square formula
    1. First let us find a perfect square \( (x + z)^2 \) which gives us : \(x^2 + bx \). This can be done by manipulating the perfect square identity:
      2. $$ (A + B)^2 = A^2 + 2AB + B^2 \\ (x + b)^2 = x^2 + 2bx + b^2 \\ (x + \dfrac{1}{2}b)^2 = x^2 + bx + \dfrac{1}{4}b^2 = x^2 + bx + (\dfrac{b}{2})^2. $$
    2. This now means that we have found a completed square formula for the above term:
      3. $$ x^2 + bx = (x + \dfrac{b}{2})^2 - (\dfrac{b}{2})^2 $$
    3. Now let us rewrite this equation into the same form using the identity above: \[ x^2 + bx + c = (x + \dfrac{b}{2})^2 - (\dfrac{b}{2})^2 + c\].
    4. Finally, lets rearrange the quadratic equation into a form where we can use the above identity we found.
      5. $$ ax^2 + bx + c \\ a (\dfrac{ax^2 + bx}{a}) + c = a(x^2 + \dfrac{b}{a}x) + c \\ $$
    5. Now we can apply the identity from above to arrive at the completing the square formula:
      6. $$ a (x + \dfrac{1}{2} \times \dfrac{b}{a})^2 - (\dfrac{1}{2} \times \dfrac{b}{a})^2 + c \\ ax^2 + bx + c = a(x + \dfrac{b}{2a})^2 + (c - \dfrac{b^2}{4a}) $$
    Derivation of Quadratic Formula
    1. Start by completing the square (see the above proof for deriving the completed square formula):
      2. $$ ax^2 + bx + c = a(x + \dfrac{b}{2a})^2 + c - \dfrac{b^2}{4a} $$
    2. Now all we have to do is use rules of algebra to rearrange the expression to be in terms of x:
      3. $$ \dfrac{b^2}{4a} - c = a(x + \dfrac{b}{2a})^2 \\ \dfrac{b^2 - 4ac}{4a} \times \dfrac{1}{a} = \dfrac{a(x + \dfrac{b}{2a})^2}{a} \\[1em] \sqrt{\dfrac{b^2 - 4ac}{4a^2}} = \sqrt{(x + \dfrac{b}{2a})^2} \\[1em] x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a} $$
    Why is there \( \pm \) before the square root? Well every square root can be either the positive square root or negative square root. For example: \( \sqrt{9} = \pm 3\).
    Warm-up question solutions
    1. An even integer n is a multiple of 2 so can be written as \(2k\). Let m represent any integer.
      \(nm = 2km = 2(km)\).

    2. Let \( 2j + 1 \) represent an odd number, n, and let \(2k + 1\) represent another distinct odd number, m.
      \(nm = (2j + 1)(2k + 1) = 4jk + 2j + 2k + 1 = 2(2jk + j + k) + 1\). By substitution of \( 2jk + j + k = z\)
      \(nm = 2z + 1\) which proves the statement.

    3. \( n^2 - n = n \times n - n = n(n - 1) \). Firstly let us prove the statement for odd numbers then even numbers. If n is odd then let \( n = 2k + 1\):
      \(2k+1(2k + 1 - 1) = 2k+1(2k)\). Since anything multiplied by an even number is an even number we can stop here. If n is even then let \(n = 2m \):
      \(2m(2m - 1).\) The same logic applied above can be used here. QED.

    4. We can disprove this given that 2 and 3 are both consecutive prime numbers and sum to an odd number; 5.

    5. Again, we can also disprove this statement given that \( a > 0\) and if we let a be 1 it is less than 2.
